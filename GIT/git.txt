git은 폴더 단위로 관리

pwd : print working directory
ls -a : 숨긴파일 포함 list
mkdir : make directory
touch : 파일 만들기

git init : 초기화(.git 폴더 생성, master)
git status : 상태확인
git add (target) : include(Staging Area에 올림)
git commit -m "first commit" (Local Repo에 올림)

git config --global : 전역변수 설정
git config --local : 지역변수 설정

git config --global user.email "cooluks2@gmail.com"
git config --global user.name "cooluks2"

git diff : 변화사항 확인

git log : 기록 확인
git log --oneline :
